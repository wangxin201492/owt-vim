### main action

#### `install`
install owvim

1. `backup`
2. `sync`
3. `addon-install`

#### `unintall`
uninstall owvim

#### `sync`
sync vimrc file from owt module to local

#### `resync`
sync ~/.vimrc from local to owt module

#### `addon-install`
install all vim addon

#### `backup`
backup ~/.vimrc
